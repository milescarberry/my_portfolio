About Me


As a data analyst, I extract valuable and comprehensible insights from data. My goal is to improve results, make the right decisions and save costs.

SQL, Python and Tableau are the core skills that I use in my data projects. Basically, I like slicing, dicing & wrangling data using Pandas and portraying my findings using the combination of Plotly & Streamlit or Tableau software.

I also have a bit of experience in machine learning, mostly with unsupervised algorithms.

I was introduced to this field in early 2020 when I started learning Google BigQuery. Afterwards, I moved on to MySQL, querying data and then visualizing my findings on Microsoft Excel. A couple of months later I started learning Python and then the libraries which were specific to data analysis and machine learning such as Numpy, Pandas, Matplotlib, Seaborn, Plotly and Scikit-Learn.

In 2022, I also added Tableau to my skillset.
